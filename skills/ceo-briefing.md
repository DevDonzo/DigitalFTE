# CEO Briefing Agent Skill

**Skill ID**: `ceo-briefing`
**Type**: Autonomous Reporting / Business Intelligence
**Tier**: Gold
**Dependencies**: Vault data, audit logs

## Purpose

Generate weekly autonomous CEO briefings with business metrics, financial summaries, bottleneck detection, and proactive suggestions.

## Architecture

```
Scheduled Trigger (Monday 8:00 AM)
    |
Weekly Audit reads:
  |-- Business_Goals.md (KPIs, targets)
  |-- Accounting/Current_Month.md (revenue)
  |-- Done/ folder (completed tasks)
  |-- Logs/*.json (audit trail)
    |
Analysis generates:
  |-- Revenue vs targets
  |-- Task completion rate
  |-- Bottleneck detection
  +-- Cost optimization suggestions
    |
Creates Briefing file:
  vault/Briefings/YYYY-MM-DD_briefing.md
```

## Implementation

### Weekly Audit Script
- **File**: `scripts/weekly_audit.py`
- **Function**: `generate_ceo_briefing()`
- **Schedule**: Monday 8:00 AM via launchd

### Schedule Configuration
- **File**: `scripts/schedule_ceo_briefing.plist`
- **Install**: `cp scripts/schedule_ceo_briefing.plist ~/Library/LaunchAgents/`
- **Load**: `launchctl load ~/Library/LaunchAgents/schedule_ceo_briefing.plist`
