# Privacy Policy

**DigitalFTE Autonomous AI Employee System**

## Overview
DigitalFTE is an autonomous AI system designed for personal and business use. This privacy policy explains how we handle data.

## Data Collection
- Emails from Gmail (read-only, for processing)
- Social media credentials (for authorized posting)
- Business data from connected services (Xero, etc.)

## Data Storage
- All data is stored **locally** on the user's machine
- No data is transmitted to external servers except:
  - Gmail API calls (encrypted, OAuth protected)
  - Social media API calls (Twitter, Facebook, Instagram)
  - Xero accounting calls (encrypted, OAuth protected)

## Data Usage
- Email data is used only for processing and action creation
- Social media posts are created as directed by the user
- No data is shared with third parties
- No data is used for training or analytics

## Security
- All API communications use OAuth 2.0 or equivalent
- No credentials are stored in plain text
- Access tokens are stored securely in local configuration

## User Rights
- Users have full control over their data
- Users can delete all local data at any time
- Users can revoke API access at any time

## Contact
For questions about this privacy policy, contact the system administrator.
