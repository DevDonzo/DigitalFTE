# Gold Tier Specification

This document is the source of truth for Gold readiness.

## Requirements

- G1: Local-first vault with dashboard and handbook
- G2: 2+ watchers (Gmail + WhatsApp)
- G3: HITL approval flow for sensitive actions
- G4: MCP servers for Email, Xero, Meta Social, Twitter
- G5: Weekly CEO briefing generation
- G6: Error recovery + audit logging
- G7: Documentation and lessons learned
- G8: Agent Skills implemented for all core actions
- G9: Cross-domain workflows (personal + business)
- G10: Browser MCP is optional/placeholder (no automation in this build)
